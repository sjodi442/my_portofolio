## 🛠️ Local Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## 📦 Building for Production

To create a production version of the app:

```bash
npm run build
```

You can preview the production build with `npm run preview`.
