# My Portfolio

A high-performance, single-page portfolio website built with SvelteKit and Tailwind CSS, featuring a "Nocturnal Brutalist" design system.

## ✨ Featured Projects

- **[Dusk Album](https://album.sjodi442.workers.dev/)** — A curated archive of visual stories exploring silence, light, and metropolitan life.
- **[LPK Saindo Web & Blog](https://lpksaindoweb.sjodi442.workers.dev)** — Official website and blog platform for LPK Saindo.
- **[Auto Kwitansi AI](https://autokwitansi.sjodi442.workers.dev)** — AI-powered online receipt generator.
- **[BenkyouLab Homepage](https://benkyoulab.online)** — Landing page for the BenkyouLab online learning platform.
- **[BenkyouLab E-Learning](https://portal-benkyou.web.id)** — Comprehensive E-Learning portal for BenkyouLab.
- **[LPK Students Management](https://lpk-students-management.vercel.app/)** — Comprehensive student management web application for LPK.

## 🚀 Built With

- [SvelteKit](https://kit.svelte.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Vite](https://vitejs.dev/)

## 🎨 Design System: Nocturnal Brutalist

This portfolio employs a custom design system characterized by:
- **Dark-first aesthetic**: Deep blacks and high-contrast surfaces.
- **Industrial Typography**: Utilizing font families like Epilogue and Grotesk for a bold, technical feel.
- **Vibrant Accents**: Neon-inspired primary, secondary, and tertiary color palettes.
- **Brutalist Elements**: Sharp edges, heavy borders, and deliberate shadows.

## 🛠️ Local Development

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

## 📦 Building for Production

To create a production version of the app:

```bash
npm run build
```

You can preview the production build with `npm run preview`.
